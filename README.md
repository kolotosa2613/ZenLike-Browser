# How To Apply The Code
1. Download the zip files from the repository dan unzip the files
2. Go to your firefox, in the URL bar type about:porfiles, in the root direcotory click open profiles
3. Then, in the directory create a files named chrome, go to downloads directory and copy all the files that have been unzipped before then paste it in the file chrome
4. Next, restart yout firefox
5. Restart your firefox and the browser should change witch some customization like the preview.
